Integración del Navegador DS 22/2025 en paiscircular.cl

Tres opciones para insertar el navegador en una nota o página de WordPress (con WPBakery). La opción A es la recomendada.

Opción A — Iframe (recomendada, ~2 minutos)

En cualquier página o nota de paiscircular.cl, insertar un bloque WPBakery → Raw HTML con:

<div style="max-width:820px;margin:0 auto">
  <iframe
    src="https://navegador-ds22.netlify.app"
    style="width:100%;min-height:800px;border:none;border-radius:12px"
    title="Navegador DS 22/2025 — Pilas y AEE"
    loading="lazy">
  </iframe>
</div>

Para que la altura se ajuste automáticamente al contenido (sin scrollbar interno), agregar este script después del iframe:

<script>
window.addEventListener('message', function(e) {
  if (e.data && e.data.type === 'resize') {
    document.querySelector('iframe[src*="navegador-ds22"]').style.height = e.data.height + 'px';
  }
});
</script>

Opción B — Link destacado (~1 minuto)

Si se prefiere no embeber, insertar un bloque de texto/HTML con un link destacado:

<a href="https://navegador-ds22.netlify.app" target="_blank" rel="noopener"
   style="display:block;padding:20px;background:#1B4332;color:#fff;border-radius:12px;
   text-decoration:none;text-align:center;font-family:sans-serif;margin:20px 0">
  <div style="font-size:11px;letter-spacing:0.05em;opacity:0.7;margin-bottom:6px">
    HERRAMIENTA INTERACTIVA
  </div>
  <div style="font-size:18px;font-weight:700;margin-bottom:4px">
    Navegador DS 22/2025 — Pilas y AEE
  </div>
  <div style="font-size:13px;opacity:0.8">
    Metas, obligaciones, plazos y exclusiones → Explorar
  </div>
</a>

Opción C — Subdominio (requiere DNS)

Si País Circular quiere servir el navegador en datos.paiscircular.cl:

  1. En Netlify: Settings → Domain management → Add custom domain → datos.paiscircular.cl.
  2. En el DNS de paiscircular.cl: agregar un registro CNAME datos → navegador-ds22.netlify.app.
  3. Esperar propagación DNS (~30 minutos). Netlify gestiona el certificado SSL automáticamente.

Activar Cloudflare Web Analytics

Analytics gratuita, sin cookies, sin banner GDPR. Pasos para activarla:

  1. Crear cuenta en https://www.cloudflare.com/web-analytics/.
  2. Agregar el sitio navegador-ds22.netlify.app (o el dominio custom si se configuró Opción C).
  3. Copiar el token que entrega el panel.
  4. En el repo, reemplazar __CLOUDFLARE_TOKEN__ en index.html por el token real.
  5. Commit + push: Netlify rebuildeará y la analítica empezará a llegar al panel.

Importante: hasta que se reemplace el token, el script de Cloudflare se carga pero no envía datos. No hay impacto en performance.

Activar Netlify Forms (newsletter + auspicio)

Los formularios están declarados en index.html y Netlify los detecta automáticamente al primer deploy.

  1. Tras el primer deploy, ir a Netlify → Site → Forms. Deberían aparecer newsletter y auspicio.
  2. Configurar notificaciones por email en Forms → Form notifications hacia info@paiscircular.cl (o quien corresponda).
  3. El plan gratuito incluye 100 envíos/mes. Si se supera, considerar upgrade o exportar regularmente.
  4. Los envíos llegan también al panel de Netlify, exportables como CSV.

Nota anti-spam: los formularios usan honeypot (bot-field). Si llegan muchos envíos basura, activar también el captcha de Netlify en Forms → Settings.

Inventario de espacios sponsor

Posiciones disponibles hoy en el navegador:

  • Banner superior (verde, sticky): tier principal, 1 sponsor único.
  • Directorio de Sistemas de Gestión (sección Obligaciones): fichas de TRAEE, Wee Chile y slots disponibles. Cada slot = 1 sponsor.
  • Webinar destacado (sección Contexto): patrocinio de la grabación o evento futuro.
  • Newsletter «Alertas DS 22/2025» (sección Contexto): patrocinio de envíos mensuales.
  • Patrocinador por sección (Metas / Plazos / Exclusiones): logo + claim al pie de cada vista.

Toda solicitud entra por el botón «Quiero auspiciar» del banner, que abre el formulario auspicio y registra el lead en Netlify Forms.
