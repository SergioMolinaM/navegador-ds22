Integración del Navegador DS 22/2025 en paiscircular.cl

Tres opciones para insertar el navegador en una nota o página de WordPress (con WPBakery). La opción A es la recomendada.

Opción A — Iframe (recomendada, ~2 minutos)

En cualquier página o nota de paiscircular.cl, insertar un bloque WPBakery → Raw HTML con:

<div style="max-width:820px;margin:0 auto">
  <iframe
    src="https://navegador-ds22.netlify.app"
    style="width:100%;min-height:800px;border:none;border-radius:12px"
    title="Navegador DS 22/2025 — Pilas y AEE"
    loading="lazy">
  </iframe>
</div>

Para que la altura se ajuste automáticamente al contenido (sin scrollbar interno), agregar este script después del iframe:

<script>
window.addEventListener('message', function(e) {
  if (e.data && e.data.type === 'resize') {
    document.querySelector('iframe[src*="navegador-ds22"]').style.height = e.data.height + 'px';
  }
});
</script>

Opción B — Link destacado (~1 minuto)

Si se prefiere no embeber, insertar un bloque de texto/HTML con un link destacado:

<a href="https://navegador-ds22.netlify.app" target="_blank" rel="noopener"
   style="display:block;padding:20px;background:#1B4332;color:#fff;border-radius:12px;
   text-decoration:none;text-align:center;font-family:sans-serif;margin:20px 0">
  <div style="font-size:11px;letter-spacing:0.05em;opacity:0.7;margin-bottom:6px">
    HERRAMIENTA INTERACTIVA
  </div>
  <div style="font-size:18px;font-weight:700;margin-bottom:4px">
    Navegador DS 22/2025 — Pilas y AEE
  </div>
  <div style="font-size:13px;opacity:0.8">
    Metas, obligaciones, plazos y exclusiones → Explorar
  </div>
</a>

Opción C — Subdominio (requiere DNS)

Si País Circular quiere servir el navegador en datos.paiscircular.cl:

  1. En Netlify: Settings → Domain management → Add custom domain → datos.paiscircular.cl.
  2. En el DNS de paiscircular.cl: agregar un registro CNAME datos → navegador-ds22.netlify.app.
  3. Esperar propagación DNS (~30 minutos). Netlify gestiona el certificado SSL automáticamente.
